# Target Brazil E-Commerce SQL Analytics

## Overview
Analyzed 100K+ e-commerce orders from Target Brazil to identify trends in sales, customer distribution, logistics performance, freight costs, and payment behavior.

## Tools
- Google BigQuery
- SQL

## Key Analyses
- Order Trends
- Customer Distribution
- Freight Cost Analysis
- Delivery Performance
- Payment Analysis

## Key Findings
- Orders increased significantly from 2017 to 2018.
- São Paulo had the highest customer concentration.
- Afternoon was the most common purchase period.
- Certain states showed significantly higher freight costs and delivery times.
